# Nonprofit CRM Reporting & Data Quality Automation Dashboard

## Overview

This project simulates a nonprofit fundraising CRM system and demonstrates how donor, gift, prospect, event, and fundraiser interaction data can be analyzed to support fundraising operations.

The project includes SQL reports, a Streamlit dashboard, data quality checks, and SOP-style documentation.

## Purpose

The goal of this project is to show how process improvement, reporting, and data quality automation can help nonprofit teams make better decisions and reduce manual work.

## Features

- Mock CRM-style donor database
- SQL reports for campaign performance, top donors, prospect research gaps, event attendance, and fundraiser activity
- Streamlit dashboard for stakeholder-friendly reporting
- Automated prospect data quality checks
- SOP and data dictionary documentation

## Tools Used

- uv
- Python
- Pandas
- SQLite
- SQL
- Streamlit

## How to Run

1. Install dependencies:

```bash
pip install pandas streamlit
```
Or 
```bash
uv add pandas streamlit 
```
2. If using uv: 
```bash
uv run streamlit run dashboard/app.py
```
